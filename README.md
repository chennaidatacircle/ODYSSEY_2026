## About ODYSSEY 2026

ODYSSEY 2026 is a flagship community-driven technology and data event organized by Chennai Data Circle, bringing together industry leaders, practitioners, researchers, students, and enthusiasts across:

* Data Engineering
* Artificial Intelligence & Machine Learning
* Analytics & Visualization
* Open Source Technologies
* Developer Communities
* Emerging Technologies

The event aims to foster collaboration, learning, innovation, and networking within the tech ecosystem.
